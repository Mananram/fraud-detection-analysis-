# Credit Card Fraud Detection Analysis

## Project Overview
This project analyses financial transaction data to identify fraud patterns and build a machine learning model capable of detecting fraudulent transactions.

## Dataset
The dataset contains 10,000 simulated credit card transactions with features such as:
- transaction amount
- merchant category
- foreign transactions
- device trust score
- transaction velocity

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Analysis Performed
- Fraud distribution analysis
- Transaction amount comparison
- Merchant category risk analysis
- Device trust score evaluation
- Transaction velocity analysis

## Machine Learning Model
A Random Forest Classifier was trained to detect fraudulent transactions.

### Model Performance
- Accuracy: **99%**
- Precision (Fraud): **1.00**
- Recall (Fraud): **0.65**
- F1 Score: **0.78**

## Conclusion
The model successfully identifies fraudulent transactions with high accuracy. Future improvements could include balancing the dataset and testing additional machine learning algorithms to improve fraud recall.
